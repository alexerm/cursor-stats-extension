Privacy Policy

Last updated: July 12, 2025

1. Information We Do Not Collect

  • The Extension does not collect or transmit any personal data (e.g., name, email, IP address).
  • No personally identifiable information is logged or stored locally or remotely.

2. Information We Access

  • The Extension injects a content script on pages under cursor.com.
  • It fetches analytics and usage data from the Cursor API using your existing session.
  • Only performance metrics and usage statistics you already have access to are displayed.

3. Use of Data

Data is used solely to render charts and statistics. No data is modified, shared, or sold.

4. Permissions

  • Host permission *://cursor.com/*: to retrieve analytics data.

5. Third-Party Services

The Extension uses only the Cursor API. No other third-party services or analytics are used.

6. Data Retention

No data is stored beyond the current browser session. All data is fetched live on each load.

7. Security

  • All requests are made over HTTPS.
  • Strict Content Security Policy: script-src 'self'; object-src 'self'.

8. Changes to This Policy

We may update this Privacy Policy at any time. Updates will be posted here with a revised date.

9. Contact

Questions? Open an issue on GitHub.
